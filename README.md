# Mobile-Price-Range-prediction-ml
📱 Mobile Price Range Prediction (Machine Learning)

📌 Overview

This project focuses on predicting the price range of mobile phones based on their features using machine learning techniques. It helps classify phones into different price categories, enabling better market analysis and product positioning.

---

🎯 Objective

- Build a classification model to predict mobile price range
- Analyze how different features affect pricing
- Provide insights for pricing strategy

---

🧠 Tech Stack

- Programming Language: Python
- Libraries: Pandas, NumPy, Matplotlib, Seaborn
- Machine Learning: Scikit-learn

---

📂 Dataset

The dataset includes mobile specifications such as:

- Battery Power
- RAM
- Internal Memory
- Screen Size
- Camera Quality
- Processor Speed

---

⚙️ Project Workflow

1. Data Preprocessing
   
   - Handling missing values
   - Feature scaling
   - Encoding 

2. Exploratory Data Analysis (EDA)
   
   - Feature distribution
   - Correlation analysis
   - Outlier detection

3. Feature Engineering
   
   - Selecting important features
   - Transformations

4. Model Building
   
   - Logistic Regression
   - Decision Tree
   - Random Forest
   - knn
   - svm

5. Model Evaluation
   
   - Accuracy
   - Precision, Recall, F1-score
   - Confusion Matrix

---

📊 Results

- Model achieved good classification performance
- Identified key features influencing mobile pricing
- acheived accuracy of 97%

---

📈 Key Insights

- RAM and battery power strongly impact price range
- High-end features lead to higher price categories
- Some features have less influence than expected
